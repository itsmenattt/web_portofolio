# Nadia Anatashiva - Portfolio Website

Website portofolio interaktif dan modern yang dibangun khusus untuk menampilkan pencapaian akademik, proyek perkuliahan, riwayat organisasi, serta tingkat kompetensi rekayasa perangkat lunak saya sebagai mahasiswa Teknik Informatika di Universitas Pembangunan Nasional.

## 🚀 Fitur Utama

- **Katalog Proyek Kuliah**: Menyajikan daftar proyek akademik lengkap dengan kategori (Web Dev, Mobile Dev, Data Science, UI/UX & SysDev), peran, detail teknis, metrik performa, dan tautan repositori/demo.
- **Riwayat Pengalaman Organisasi**: Menampilkan lini masa keaktifan organisasi dilengkapi deskripsi tugas serta pencapaian nyata secara terukur.
- **Matriks Kompetensi & Akademik**: Visualisasi penguasaan instrumen teknologi (Frontend, Backend, DevOps, Data Science) serta transkrip nilai mata kuliah pemrograman inti.
- **Formulir Hubungi Secara Profesional**: Antarmuka interaktif bagi perekrut atau kolaborator untuk mengirimkan proposal proyek langsung secara asinkronus.

## 🛠️ Tech Stack

Aplikasi ini dirancang menggunakan standar teknologi web modern berkinerja tinggi:

- **Core**: React 19 (TypeScript) & Vite
- **Styling**: Tailwind CSS & Vanilla CSS
- **Animasi & Transisi**: Motion for React
- **Ikonografi**: Lucide React

## 💻 Cara Menjalankan Secara Lokal

Ikuti langkah-langkah berikut untuk menjalankan website portofolio ini di komputer Anda:

### Prasyarat

Pastikan Anda telah menginstal **Node.js** (rekomendasi versi 18 ke atas) di sistem Anda.

### Langkah-Langkah

1. **Unduh atau Kloning Repositori**
   ```bash
   git clone https://github.com/itsmenattt/web_portofolio.git
   cd web_portofolio
   ```

2. **Instalasi Dependensi**
   ```bash
   npm install
   ```

3. **Menjalankan Server Pengembangan**
   ```bash
   npm run dev
   ```
   Buka [http://localhost:3000](http://localhost:3000) di browser Anda untuk melihat hasilnya secara langsung.

4. **Membuat Build Produksi**
   ```bash
   npm run build
   ```
   Build hasil kompilasi siap sebar akan tersimpan di dalam direktori `dist/`.
